# 📋 Kanban Task Management Application

**Live Demo:** ##  (https://kanban-task-management-application-woad.vercel.app/)

## 📝 About The Project
This is a single-page Kanban Task Management web application developed It provides a seamless, interactive interface for users to create, manage, and organize tasks across different workflow.

The application is built to be lightweight and fast, featuring a custom glassmorphism UI and persistent local storage so your board state remains perfectly intact even after a page refresh.

## ✨ Features
This project successfully implements all core requirements and optional enhancements from the task prompt:

* **Interactive Kanban Board:** Default columns for *Todo*, *In Progress*, and *Done*
* **Native Drag & Drop:** Smoothly drag tasks between columns with immediate UI updates.
* **Task Management:** Create, edit, and delete tasks with titles and descriptions.
* **State Persistence:** Task data is saved automatically to the browser's `localStorage` and survives page refreshes
* **Fully Responsive:** Beautifully designed to function properly on both desktop and mobile screen.
* **🟢 Priority Tags (Enhancement):** Assign High, Medium, or Low priorities to tasks.
* **🔍 Search & Filter (Enhancement):** Instantly search through tasks by title or description.
* **🌗 Theme Toggle (Enhancement):** Switch between a sleek dark mode and a clean light mode.

## 🚀 How to Run Locally
Because this project is built using native web technologies and a CDN, running it locally requires zero installation or build tools:
1. Clone this repository or download the ZIP file.
2. Ensure all files (`index.html`, `style.css`, `app.js`, `background.jpg`) are extracted into the same folder.
3. Double-click `index.html` to open and run the application in any modern web browser.
